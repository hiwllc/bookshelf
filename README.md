# MinhaCompra
> API para criar listas de compras

## Road Map
- Poder criar conta (epic - users).
- Poder criar uma lista de compra com nome, descrição e uma imagem.
- Poder adicionar produtos nesta lista com nome, link, preço, quantidade e imagem.
- Na lista de compra mostrar o valor total da quantidade de produto, em detalhes de produto mostrar o valor por quantidade.
- Na lista de compra ver o valor total.
- Compartilhamento de listas
